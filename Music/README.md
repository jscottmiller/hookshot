# Music

This directory should contain background music tracks.

The original game used licensed Pond5 audio that cannot be redistributed. You'll need to add your own music files here.

## Expected Files

- Background loop music (WAV or OGG format)

## Suggested Free Sources

- [Freesound](https://freesound.org)
- [OpenGameArt](https://opengameart.org)
- [Free Music Archive](https://freemusicarchive.org)
